Import all your sound assets to this folder to use with Resound.
'Playlist' class will be created (or updated) after you import any sound assets.

Example usage with Playlist: 
	Resound.PlaySFX(Playlist.Click);

Or use plain string:
	Resound.PlayMusic("MainMenu");

*Note:
Doesn't work with subfolder. So put all your sounds in this folder!
